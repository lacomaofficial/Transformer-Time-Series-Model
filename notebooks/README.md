# Experimental Models

This repository contains a collection of experimental notebooks exploring various approaches to time series analysis, ranging from traditional deep learning architectures to novel Large Language Model (LLM) integrations.


1.  **Univariate vs. Multivariate:** Comparing single-variable forecasting against multi-variable systems that capture inter-dependencies.
2.  **Classification vs. Forecasting:** Distinguishing between predicting a future value (regression) and identifying a state or direction (classification).
3.  **LLM Adaptation:** How to bridge the gap between discrete text tokens and continuous time series data.
4.  **Financial Data Nuances:** Handling specific challenges like volatility, missing data, and market hours in Yahoo Finance datasets.
5.  **Architecture Comparison:** Seeing how standard Deep Learning models stack up against emerging LLM-based approaches.
